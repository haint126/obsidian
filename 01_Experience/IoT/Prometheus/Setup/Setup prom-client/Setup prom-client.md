# Pre-requisite
- Install:
	- node
	- npm
	- git

# Install prom-client
- Clone prom-client project

```bash
git clone https://github.com/siimon/prom-client.git
```

- Go to prom-client folder
- Install `express`

```bash
npm install express
```

- Install `prom-client`

```bash
npm install prom-client
```

- Run demo program:

```bash
node ./example/server.js
```
