# A machine learning workflow
1. Define the problem
1. Assemble a dataset
1. Determine success metrics
1. Decide on evaluation method
1. Prepare the data
1. Establish a baseline
1. Beat the baseline
1. Deploy the model
1. Monitor it
1. Collect feedback
1. Rebuild
