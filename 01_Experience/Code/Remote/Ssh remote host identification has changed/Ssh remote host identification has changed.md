# Error: Remote host identification has changed
## Error message
```bash
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
```

## Fix
```bash
ssh-keygen -R <host>
```
